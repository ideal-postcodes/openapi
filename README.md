## Getting Started

### Build Typings

```
npm run build
```

### Validate Spec

```
npm test
```

### Download and Access via npm

#### Install

```bash
npm install @ideal-postcodes/openapi
```

#### Require specs

JSON and YAML specs can be accessed at `node_modules/@ideal-postcodes/openapi/dist/openapi.[json|yaml]`
